What You're Looking At — How "Ass-in-Chair" Time Is Measured

This report measures one thing: how many hours each BDR was actively sitting at their desk and dialing. It is not a call count. It is not talk time. It is active dialing session time — the window during which a rep is on the phones.

The 20-Minute Gap Rule

Every call has a timestamp. The calculation works like this:

  • The first outbound call of the day opens a dialing block.
  • Each next call — outbound or inbound — keeps the block open as long as it happens within 20 minutes of the previous one.
  • If the gap exceeds 20 minutes, the block closes. Only the next outbound call opens a new block — an inbound call after a gap does not.
  • AIC time = total minutes across all blocks for the day.

Worked Example

9:00 AM →9:14 AM →9:28 AM ✗ 31-min gap ✗9:59 AM →10:11 AM

Block 1: 9:00 → 9:28 = 28 min  | Block 2: 9:59 → 10:11 = 12 min  | Total AIC = 40 min

Avg / Workday vs Avg / Active Day

Avg / Active Day = average only on days they actually dialed. Avg / Workday = average spread across every Monday–Friday in the period, including days they didn't dial at all. The second number is the harder benchmark.
